# I-test

Quotes web-crawler.

## Getting Started

### Prerequisites

- RVM
- Ruby 2.6.5
- MongoDB 4.2.4

### Installation

Create a `.env` file in root and set following variables:

```
DATABASE_HOST=<value> (eg.:localhost:27017)
```

Install dependencies:

```
bundle
```

## Development

Running locally:

```
rails s
```

Running tests:

```
rails test
```
